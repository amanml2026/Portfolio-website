# Personal Portfolio Website

A clean and responsive personal portfolio website built using **HTML**, **CSS**, and **Bootstrap** to showcase my projects, skills, and learning journey in AI & Machine Learning.

## 🌐 Live Website

https://amanml2026.github.io/Portfolio-website/

## ✨ Features

- Responsive and modern design
- Home, About, Projects, and Contact pages
- Clean and minimal user interface
- Beginner-friendly project structure

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome

## 📂 Project Structure

```
Portfolio-website/

│── projects.html

│── about.css
│── projects.css

```

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/amanml2026/Portfolio-website.git
```

2. Open `index.html` in your browser.
